# Alzheimer’s Disease Classification using Hybrid CNN & SOM
This repository contains a machine learning framework for Alzheimer’s Disease classification using deep learning (Convolutional Neural Networks) and unsupervised learning (Self-Organizing Maps).

This project was developed as part of a Master’s thesis at the University of Jos, focusing on automated diagnosis from brain imaging data and improving interpretability through visualization techniques.

## Contents
* Jupyter/Colab notebooks for training and evaluation
* CNN and SOM model implementation
* Visualization outputs (confusion matrix, ROC curve, performance plots)
* Processed results and figures

## Key Features
* CNN-based brain image classification
* Self-Organizing Map (SOM) for feature clustering and visualization
* Performance evaluation using Accuracy, Precision, Recall, and F1-score
* ROC Curve and Confusion Matrix analysis
* Clear visual representation of model results

## Results
All results are provided as static images for proper GitHub rendering, including:
